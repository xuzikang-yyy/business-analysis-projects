- 输出可视化图表与业务优化建议

### 2. 企业财务成本与利润分析
- 分析产品线收入、成本、利润率
- 计算盈亏平衡点，提供成本优化方案
- 贴合财务专业，展示商业理解能力

### 3. 运营数据自动化报表
- 自动读取数据 → 计算指标 → 生成图表
- 提高报表制作效率，适合日常运营监控
